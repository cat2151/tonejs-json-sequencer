# issue demo streaming の見た目を改善する #110
[issues #110](https://github.com/cat2151/tonejs-json-sequencer/issues/110)

# 詳細
- dark modeで「ヒント」欄の背景が明るい黄色のため、白い文字色と重なって何も見えない。dark modeのヒント欄の背景の「明るい黄色」を、例えば「文字色の白と見分けがつく、暗い色」に変更する
- 「このデモについて」欄は、デフォルトで折りたたみする。「このデモについて」ボタンを押すと折りたたみ解除する
- 「使い方」欄は、デフォルトで折りたたみする。「使い方」ボタンを押すと折りたたみ解除する
- 「使い方」欄を、「このデモについて」欄のすぐ下に移動する
