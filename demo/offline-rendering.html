How to use:

  1. Select a sequence from the dropdown or edit the NDJSON in the textarea
  2. Adjust rendering configuration if needed (sample rate, end buffer, filename)
  3. Click "Render Audio" to start offline rendering
  4. Preview the rendered audio in the player
  5. Click "Download WAV" to save the audio file

Note: Offline rendering is faster than real-time and produces perfect timing!
